ćwiczenia z interpolacji

1. Wypisz kolejno pola z klasy: 'title', 'num1', 'num2', 'falsity'

to naprawdę fajne {{ title }}

a imię jego {{ num1 }} i {{ num2 }}

True jest przeciwieństwem {{ falsity }}

2. Wypisz pola z obiektu 'dog'

mój pies ma na imię: {{ dog.name }}

ma lat: {{ dog.age }}

jest koloru: {{ dog.color }}

3. Wypisz kolejno quotation1 i quotation2 z klasy komponentu:

Cytat 1: {{ quotation1() }}

Cytat 2: {{ quotation2 }}

4. Stwórz wyrażenia:

Dodaj dwie cyfry, odejmij i pomnóż.

Wyrażenie: ...

Użyj skróconego if-a i w zależności od gettera 'randomBoolean' jeżeli jest true wyświetl 'tak' jeżeli false 'nie';

Wynik: {{ randomBoolean ? 'tak' : 'nie' }}

Podziel zmienną 'num1' przez metodę 'divider': {{ floorDivide(num1) }}

Wynik: {{ floorDivide(num1) }}
